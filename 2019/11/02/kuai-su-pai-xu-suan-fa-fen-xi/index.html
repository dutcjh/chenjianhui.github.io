快速排序算法分析

快速排序算法有时候也叫“快速排序”、“快排”，是在冒泡排序的基础上，增加分治策略而得到的一种用于对数字进行排序的算法。快速排序的大致过程是这样：(1)我们把大于基准值的数字移动到基准值的右边(在原始数组上进行这个操作)，把小于基准值的数字移动到基准值的左边，这样就确定了基准值的位置，于此同时，整个数组被划分为两个不可能有交叉的部分（基准值右边的元素比基准值左边的元素大）；(2)接下来，我们分别对基准值左边和右边的小数组进行前面的操作；(3)递归地进行，直到小数组里只有一个数字。

Python代码实现：

def quickSort(arr, left = None, right = None):
    left = 0 if not isinstance(left, (int, float)) else left
    right = len(arr)-1 if not isinstance(right, (int, float)) else right
    #判断low是否小于high,如果为false,直接返回
    if left < right:
        i,j = left,right
        #设置基准数
        base = arr[i]

        while i < j:
            #如果列表后边的数,比基准数大或相等,则前移一位直到有比基准数小的数出现
            while (i < j) and (arr[j] >= base):
                j = j - 1

            #如找到,则把第j个元素赋值给第个元素i,此时表中i,j个元素相等
            arr[i] = arr[j]

            #同样的方式比较前半区
            while (i < j) and (arr[i] <= base):
                i = i + 1
            arr[j] = arr[i]
        #做完第一轮比较之后,列表被分成了两个半区,并且i=j,需要将这个数设置回base
        arr[i] = base

        #递归前后半区
        quickSort(arr, left, i - 1)
        quickSort(arr, j + 1, right)
    return arr

快速排序为什么叫”快速”排序呢？因为它时间复杂度是O(nlogn)， 平均速度比较快，应用比较广，比如python的sorted函数使用的就是快速排序。另外，这个算法里用到了递归，还需要我们使用一些策略来压缩内存的消耗，算是元素比较多的一个内容，因此是教材必讲算法。
