NP问题1

从1、2、2、3、9、5、5、25这8个数中，选出2个或2个以上的数相乘，可以得到多少个不同的乘积？

方法一：针对含1的数组，双重循环求解

def getmult(arr): # 含有1的数组
    lst = {1} # 创建一个集合，含1
    for node in arr: 
        lst = lst | set([m*node for m in lst])
    return lst

方法二：双重循环求解

def getmult(arr):
    result_list = [arr[0]]
    for num in arr[1:]:
        result_list += [p*num for p in result_list]
        result_list.insert(0, num)
    return list(set(result_list[len(arr):]))

或：

def getmult(arr):
    result_list = []
    for i in range(len(arr)):
        temp = arr[:i] + result_list; # 拼接乘数数组
        result_list += [p*arr[i] for p in temp]
    return list(set(result_list))

方法三：递归方法

def getmult(arr): # 递归
    length = len(arr)
    if length < 2:
        return []
    if length == 2:
        return [arr[0]*arr[1]]
    else:
        pre = getmult(arr[:length-1]) #前一步结果
        temp = arr[:length-1] + pre # 列表合并
        last = pre + [arr[-1]*p for p in temp] #后一步结果
        return list(set(last)) #后一步结果
